Často kladené otázky

Jak dlouho trvá vytvoření webu? +
Typicky 4-8 týdnů v závislosti na rozsahu projektu a tvojí dostupnosti pro konzultace. Komplexnější projekty mohou trvat déle.
Jaké je vaše ceny? +
Ceny se liší podle rozsahu a vašich potřeb. Nabízím flexibilní balíčky od 5000 CZK. Rád bych se s vámi spojil pro detailní wycenění.
Poskytujete údržbu po spuštění? +
Ano, nabízím měsíční údržbu a podporu. Zajistím aktualizace, bezpečnost a případné úpravy.
Mohu si web sám spravovat? +
Samozřejmě. Web bude používat snadno upravovatelný CMS systém s intuitívním rozhraním. Poskytuji i školení.
